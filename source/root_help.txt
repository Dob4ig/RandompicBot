Команды для root:
$ help -- Вывести список команд.
$ ban_user {id} -- Заблокировать пользователя по id.
$unban_user {id} -- Разблокировать пользователя по id.
$get_list {role} -- Получить список пользователей по роли(ролям) {user, banned, admin}. 
$set_admin {id} -- Установить пользователю роль "admin".
$del_admin {id} -- Удалить пользователю роль "admin" (Станет user).
$msg {role} -- Рассылка пользователям из БД (Ввести роли пользователей).
